CSS Files and Structure

This template used 6 CSS files in this template. By using of them, we can work round this. Those files also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the files below:

  1. style.css: Main style
  2. css/bootstrap.css: Bootstrap Librabry
  3. css/plugin.css: all jquery plugin css file need
  4. css/fonts.css: Icon Font
  5. css/responsive.css: Custom CSS for Responsive Layout
  6. css/animate.css: CSS3 Animation Library

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Portfolio Filter

To add new or modify current categories change data-option-value attribute to category which you want.

<ul class="clearfix option-set" data-option-key="filter" id="options">
	<li class="clearfix">
		<a class="selected" data-option-value="*" href="#filter">All<span class="count">99</span></a></li>
	<li class="clearfix">
		<a data-option-value=".t1" href="#filter">Creative<span class="count">16</span></a></li>
	<li class="clearfix">
		<a data-option-value=".t2" href="#filter">Character<span class="count">9</span></a></li>
	<li class="clearfix">
		<a data-option-value=".t3" href="#filter">Illustrator<span class="count">56</span></a></li>
	<li class="clearfix">
		<a data-option-value=".t4" href="#filter">Concept<span class="count">56</span></a></li>
</ul>

Contact Form

Contact form is based on HTML 5 AJAX Contact Form. You don't have to know PHP. To change email adress, open "contact.php", find and edit code below:

    // Configuration option.
    // Enter the $email $address that you want to emails to be sent to.
    // Example $address = "joe.doe@yourdomain.com";
    $address = "mail@example.com";
